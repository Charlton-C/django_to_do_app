## Installation

Ready to get started? Follow these simple steps and youâ€™ll have the Todo app running on your local machine in no time!

1. **Clone the Repo**  
   Start by cloning the repository to your machine:

   ```
   git clone https://github.com/your-username/todo-app.git
   ```
2. **Set Up Your Virtual Environment**  
   Create a virtual environment to manage your dependencies:

   ```
   python -m venv venv
   ```
3. **Activate the Virtual Environment**  
   Activate your environment:
   - On **Windows**:

     ```
     venv\Scripts\activate
     ```
   - On **macOS/Linux**:

     ```
     source venv/bin/activate
     ```
4. **Install Dependencies**  
   Install the required Python libraries:

   ```
   pip install -r requirements.txt
   ```
5. **Migrate the Database**  
   Set up the database:

   ```
   python manage.py migrate
   ```
6. **Create a Superuser (Optional)**  
   If you want to access the Django admin panel:

   ```
   python manage.py createsuperuser
   ```
7. **Run the App**  
   Launch the development server:

   ```
   python manage.py runserver
   ```
8. **Visit the App**  
   Open your browser and go to:  
   <http://127.0.0.1:8000/>
